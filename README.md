# Mi Página HTML Semántica

Esta es una página web personal desarrollada como parte de una práctica de HTML semántico y CSS.

## Tecnologías utilizadas

- HTML
- CSS

## Contenido

La página incluye:

- Un encabezado con información personal.
- Un menú de navegación.
- Una sección de presentación.
- Una lista de intereses.
- Una imagen de perfil.
- Enlaces externos.
- Un pie de página con información de contacto.
